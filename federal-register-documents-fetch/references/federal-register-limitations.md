# Federal Register Limits

- This skill searches official Federal Register document metadata only.
- It does not fetch full HTML page bodies or parse linked PDFs.
- Publication filtering is date-based, not time-of-day-based.
- Returned `excerpts` can contain highlight markup from the service.
- Federal Register documents capture official policy publication context, not public reaction by themselves.
- Docket discussion and public comment text belong in Regulations.gov, not this skill.
